# Mahammad Akhil - Cloud / DevOps Engineer Portfolio

Welcome to my professional portfolio! This project is a fully responsive, meticulously designed single-page application built with React, Vite, and vanilla CSS. It features a stunning, Apple/GitHub-inspired pure white and black minimalist aesthetic designed to showcase my DevOps, Cloud Infrastructure, and CI/CD skills.

## 🚀 Features

- **Pure Minimalist Aesthetic**: A strict white and black color palette with elegant hover states, designed for absolute clarity and maximum contrast.
- **Interactive "Terminal" Boot Sequence**: A custom loading screen mimicking a Linux boot loader.
- **DevOps Themed UI**: 
  - Typing terminal effects in the Hero section.
  - Interactive Server Rack LED blink effects on project cards.
  - Hexagonal skill grids with glowing hover states.
  - Sudo/kubectl easter egg for tech enthusiasts.

## 💻 Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Vanilla CSS (CSS Variables, Flexbox, CSS Grid, custom keyframe animations)
- **Deployment**: Ready for Vercel, Netlify, or GitHub Pages

## 🛠️ How to Run Locally

Follow these steps to get the portfolio running on your local machine:

### Prerequisites

Ensure you have the following installed:
- [Node.js](https://nodejs.org/) (v16.0.0 or higher recommended)
- `npm` (comes with Node.js) or `yarn`

### Installation & Setup

1. **Clone the repository** (or download the source code):
   ```bash
   git clone <your-repo-url>
   cd portfolio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **View the application**:
   - Open your browser and navigate to `http://localhost:5173/` (or the port specified in your terminal).

### Building for Production

To create a production-ready build of the portfolio:

```bash
npm run build
```

This will generate an optimized bundle in the `dist` directory. You can preview the production build locally by running:

```bash
npm run preview
```
